# 🐍 Python Logic Practice (Estudos Iniciais)

Este repositório reúne exercícios e pequenos scripts desenvolvidos durante meus primeiros estudos em Python, com foco no desenvolvimento de lógica de programação.

## 📌 Sobre o Projeto

Este projeto foi criado a partir de um curso introdutório de Python realizado anteriormente, com o objetivo de consolidar fundamentos essenciais da programação.

> ⚠️ Este não é um projeto em desenvolvimento ativo. Foi mantido como registro da minha evolução na área de tecnologia.

## 🎯 Objetivo

- Desenvolver raciocínio lógico
- Aprender estruturas básicas de programação
- Praticar resolução de problemas
- Criar base sólida para estudos mais avançados

## 🚀 Tecnologias Utilizadas

- Python 3

## 🧠 Conceitos Praticados

- Variáveis e tipos de dados
- Estruturas condicionais (`if`, `else`)
- Estruturas de repetição (`for`, `while`)
- Funções
- Entrada e saída de dados
- Lógica de resolução de problemas
